样例

这些简单的应用可以帮助你开始熟悉并运行Crosswalk项目。

所有简单的资源都可以在github (MIT license)上获取。为了使用这些样例，首先需要复制该项目：

$ git clone https://github.com/crosswalk-project/crosswalk-samples.git

这样将会生成一个包含若干项目子目录的crosswalk-samples目录：

  • extensions-android – 一个简单的Crosswalk Android扩展样例。Android扩展程序教程介绍了如何开发该应用。关于编译指令，请参见页面README。

  • hello-world – 一个简单的样例。

  • simd-mandelbrot – 使用Crosswalk对SIMD原生支持的样例。

  • space-dodge-game – 一个模拟你在宇宙飞船中躲避小行星的游戏。该应用中包含为Crosswalk screen APIs开发的代码。

  • webgl – 一个关于如何在Crosswalk中通过three. js使用WebGl的样例。

  • webrtc – 一个关于在Crosswalk中使用WebRTC API的样例。开发过程详见WebRTC教程。关于编译指令，请参见页面README。

关于样例更多的信息，请点击菜单栏中的链接。
