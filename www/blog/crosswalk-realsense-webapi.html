Introducing Crosswalk Project Extensions for Intel® RealSense™ Technology

Tue, 23 Feb 2016 04:00:00 GMT by Web RealSense Team

Intel® RealSense™ Technology redefines how we interact with our devices for a more natural, intuitive and immersive experience. By introducing 3D cameras with depth information combined with regular RGB data, new use cases are possible such as augmented reality, hand/face recognition, object tracking, and more. The Crosswalk Project Extensions for Intel® RealSense™ Technology now brings these amazing capabilities to web developers.

Our first public release (part of Crosswalk v18.6.0) includes the following:

  • Depth Enhanced Photography (spec)
  • Scene Perception (spec)
  • Face Tracking and Recognition (spec)

Developers can create 3D-aware web applications that can be run on any Windows tablet with a built-in R200 camera, such as the HP Spectre x2, or a Windows PC with peripheral R200 camera.

##Getting Started Follow the quick start page to build your Intel® RealSense™ powered web application. You can also download the sample for a quick glance.

##Get involved We invite you to contribute. This project is under the Crosswalk Project umbrella. The APIs are developed as external Crosswalk extensions, and the source code is hosted at http://github.com/crosswalk-project/realsense-extensions-crosswalk. Please share any ideas you have for discussion on our mailing list: crosswalk-dev and crosswalk-help. Follow the instructions to build the project. Contribute your changes following regular github pull requests.

The Crosswalk Project bug tracking system is: https://crosswalk-project.org/jira/. For the API bugs, go to the Extensions for RealSense component.

##FAQ

####Is Windows the only platform supported? How about other platforms like Android?
Currently Windows is the only platform shipping with R200 cameras. Support for Android will be considered once hardware is available and shipping with this capability.

####Intel® RealSense™ SDK has a lot of native APIs. Why are the web APIs limited to DEP, SP and Face?
The team is working hard to add support for more capabilities. Please check the API dashboard page.
